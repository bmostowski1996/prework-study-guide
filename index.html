Prework Study Guide

HTML

  • HTML is a way of organizing webpage content.
  • The head element contains information the browser needs to "decode" the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS is how you add "style" to webpages made with HTML.

Git

  • Git is a version-control system for keeping track of code.
  • git status: Lists what branch we are in.
  • git checkout -b branch-name: creates a new branch called branch-name and switches to that branch

Javascript

  • Javascript is a way of making webpages interactable!
